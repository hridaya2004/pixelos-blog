---
title: Signing Builds
description: How to sign your official PixelOS builds using the LineageOS signing structure.
toc: false
---

PixelOS uses the LineageOS signing structure for all official builds. Use the following template to set up your signing keys:

[https://github.com/ItsVixano/android_vendor_lineage-priv_keys](https://github.com/ItsVixano/android_vendor_lineage-priv_keys)

Follow the instructions in the template to generate and place your signing keys in the expected directory structure before starting your build.

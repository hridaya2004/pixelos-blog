## Before you report

1. Search the group or forum to check if the bug has already been reported.
2. Reboot your device and check if the issue persists.
3. Do a clean flash — format data or factory reset using the recommended recovery.
4. Reproduce the bug on a stock ROM with no Magisk modules installed.
5. Use the default kernel shipped with the ROM.
6. Use the recovery recommended by the maintainer.

<Callout type="warn" title="WARNING">
  Reports from users running a modified kernel, skipping a clean flash when required, or using a
  decrypted setup may be closed without investigation.
</Callout>

## What to include

When you file your report, include the following:

- **Device variant** — the exact model or region variant of your device.
- **PixelOS version** — the build you're running when the issue occurs.
- **Logs** — see [How to capture logs](/docs/users/guide/capture-logs) for instructions.
